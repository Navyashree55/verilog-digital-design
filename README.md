Verilog Digital Design Implementations

This repository contains my Verilog implementations of digital design concepts, covering a progression from basic logic gates to memory modules such as Dual-Port RAM.
Topics Covered

    Basic Logic Gates
    Combinational Circuits (Adders, Multiplexers, Decoders, Encoders)
    Sequential Circuits (Flip-Flops)
    Memory Modules (Single-Port RAM, Dual-Port RAM)

Features

    Modular and well-organized Verilog code
    Testbenches for functional verification
    Structured progression from fundamental to advanced concepts

Tools Used

    EDA Playground (Simulation)
    GitHub (Version Control)
